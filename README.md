# Sistema de Cadastro de Usuários

Sistema CRUD completo para gerenciamento de usuários, desenvolvido com foco em organização, boas práticas e escalabilidade.

---

## Sobre o Projeto

Este projeto foi desenvolvido para realizar o gerenciamento completo de usuários, permitindo:

- Criar usuários
- Listar usuários
- Atualizar informações
- Remover usuários

A aplicação segue o padrão CRUD (Create, Read, Update, Delete), muito utilizado no desenvolvimento de sistemas web e APIs.

---

## Funcionalidades

- Cadastro de usuários
- Listagem de usuários
- Edição de dados
- Exclusão de usuários
- Validação de campos
- Interface organizada
- Integração entre front-end e back-end
